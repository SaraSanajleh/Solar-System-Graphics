# Educational 3D Solar System Simulation

## Overview

This project is an interactive 3D Solar System simulation developed using OpenGL and C++. The application visualizes the Sun, planets, moons, and orbital motion in a dynamic environment while demonstrating fundamental computer graphics concepts such as transformations, animation, texture mapping, lighting, and user interaction.

## Features

- Real-time planetary revolution around the Sun.
- Planet self-rotation animation.
- Textured planets and space background.
- Multiple camera modes for scene exploration.
- Mouse and keyboard interaction.
- Zooming and free navigation.
- Hover-based planet information display.
- Saturn ring visualization.
- Dynamic moon movement.
- Switchable lighting and shading modes.

## Technologies Used

- C++
- OpenGL
- GLUT
- GLU
- stb_image.h (Texture Loading)

## Graphics Concepts Implemented

### Transformations

- Translation for planet positioning.
- Rotation for orbital and axial motion.
- Scaling for adjusting celestial body sizes.

### Animation

- Continuous real-time motion using timer-based updates.
- Adjustable animation speed.
- Pause and resume functionality.

### Texture Mapping

- Realistic textures applied to planets, the Sun, and the surrounding environment.
- Runtime texture enable/disable option.

### User Interaction

- Keyboard controls for navigation and scene management.
- Mouse controls for camera rotation and zooming.
- Planet information display on hover.

### Lighting and Shading

- Ambient, diffuse, and specular lighting.
- Smooth and flat shading modes.
